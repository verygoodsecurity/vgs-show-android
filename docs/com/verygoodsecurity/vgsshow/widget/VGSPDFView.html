public final class VGSPDFView
extends VGSView<PDFView>

    A VGSPDFView can be used to display a PDF file.

    VGSPDFView depends on AndroidPdfViewer library. When View is initialized without additional dependency, the View will throw Exception. Please add the dependency to your module gradle file before using this View:

    implementation 'com.verygoodsecurity:android-pdf-viewer:<latest_version>'
